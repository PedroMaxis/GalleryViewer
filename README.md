
# Gallery Viewer

Uma aplicação usada para enviar uma imagem com titulo e exibir em grid mosaico.





## Rodar Localmente

Clone o projeto.

```bash
  git clone https://github.com/PedroMaxis/GalleryViewer.git.
```

### Instale o Backend

Vá para o diretorio do projeto.

```bash
  cd backend
```

Instale as dependencias.

```bash
  npm install
```

Inicie o Servidor

```bash
  node index.js
```
### Instale o Frontend
Vá para o diretorio do projeto.

```bash
  cd frontend
```

Instale as dependencias.

```bash
  npm install
```

Inicie o Servidor

```bash
  npm start
```
